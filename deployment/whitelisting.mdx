### Services
Some of the features on the app require external services to work properly. For the Actions and External Vector Database features, the URL depends on the configuration of the user.
| Service Name   | Endpoint     |
|----------------|----------------|
| Web: Bing Search    | api.bing.microsoft.com    |
| Web: Langdock Web Proxy | services.langdock.com |
| Code Interpreter | `REGION`.dynamicsessions.io    |
| Actions | Custom |
| External Vector Database | Custom |

### Model Providers
Depending on the models you want to enable in the app, you will need to whitelist the endpoints of the respective model providers.
| Model Provider   | Endpoint     |
|----------------|----------------|
| OpenAI API    | api.openai.com    |
| Azure OpenAI API    | `DEPLOYMENT-ID`.openai.azure.com    |
| Anthropic API    | api.anthropic.com or 160.79.104.0/23 and 2607:6bc0::/48    |
| AWS Bedrock    | bedrock.`REGION`.amazonaws.com	    |
| Google Vertex AI    | `REGION`-aiplatform.googleapis.com    |
| Azure AI Studio    | `DEPLOYMENT-ID`.`REGION`.models.ai.azure.com, `DEPLOYMENT-ID`.`REGION`.inference.ai.azure.com    |
