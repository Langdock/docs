## Welcome to Langdock's Documentation Page! 📘

This is the documentation of Langdock. Langdock is a GDPR-compliant AI chat application that is specifically designed for companies.

Langdock can be used in many different situations. Some examples are:
- Creating texts, including copy, emails, speeches, and presentations
- Generating and analysing images
- Working with existing files, such as summarizing, asking questions, translating, analyzing, and classifying
- Extracting knowledge from large volumes of data and various sources
- Writing and debugging code

This guide will walk you through everything you need to know about Langdock, from getting your account set up to making the most of our platform. This is the first version of our documentation, we will continuously improve it and add more examples!
