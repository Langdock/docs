---
title: 'Create and Share an API Assistant in Langdock'
description: 'The following guide explains how to create an API key in Langdock and share an assistant with the API key.'
---

<Info>An admin needs to create the API key and share the assistant with the API key. Invite an admin as an editor to your assistant with the "Share" button in the top right corner (skip this step if you are a Langdock admin yourself).</Info>


1. Navigate to [https://app.langdock.com/chat](https://app.langdock.com/chat)

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/348df815-6028-4264-9edf-c3776b7015c8/ascreenshot.jpeg?tl_px=0,0&br_px=3840,2160&force_format=jpeg&q=100&width=1120.0)


2. Navigate to the workspace settings.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/9a4155f7-9ac5-470d-9a03-ed63e61116bd/ascreenshot.jpeg?tl_px=0,0&br_px=2102,1174&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=90,81)


3. Click on "API" under products.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/879bb10a-4ae9-41d6-a586-6a6eae244d37/ascreenshot.jpeg?tl_px=0,0&br_px=3840,2160&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=33,393)


4. Create a new API key.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/c88f02bb-fb8a-4491-a3fe-fe80a8323fcc/ascreenshot.jpeg?tl_px=812,423&br_px=3840,2114&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=725,277)


5. Enter a name and click "Create API key"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/b7b15e51-326a-4738-a550-8d05a558a61e/ascreenshot.jpeg?tl_px=941,544&br_px=3043,1719&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=693,337)


6. Copy your API Key to the clipboard.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/934e809c-aba3-444e-aa99-b1b5b630dcc6/user_cropped_screenshot.jpeg?tl_px=934,549&br_px=3036,1724&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=688,290)


7. Click "Done"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/ebc6997b-cc71-4fbe-b12b-241a3d837b2a/user_cropped_screenshot.jpeg?tl_px=929,502&br_px=3031,1677&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=709,374)


8. Leave the settings by clicking on "Settings" on the top left.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/6b41e72c-0574-4c65-996b-5867ee378f78/ascreenshot.jpeg?tl_px=0,0&br_px=1610,899&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=58,21)


9. Click "Assistants"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/63c2585d-70d5-4f16-bc53-2a16ac0058c3/ascreenshot.jpeg?tl_px=0,0&br_px=1610,899&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=124,171)


10. Create a new assistant

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/d2d44547-9b23-4864-8053-77aecfb6fcba/ascreenshot.jpeg?tl_px=1737,0&br_px=3840,1174&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=530,87)


11. Configure your assistant. To save it, at least enter a name.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/7a3d2352-6016-4911-a072-3bcedd38293e/ascreenshot.jpeg?tl_px=0,0&br_px=3027,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=406,117)


12. Click "Share"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/eca0a6fb-66e7-48f2-a731-42d8af675adb/ascreenshot.jpeg?tl_px=1737,0&br_px=3840,1174&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=891,-8)


13. Search for the name of your API key.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/6b18d43c-71e0-43f1-8c05-2e4ba609eb8f/ascreenshot.jpeg?tl_px=900,467&br_px=3002,1642&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=426,249)


14. Click "Share"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/dd0f0f0a-8e2c-458f-8a93-27b8d80ef22e/ascreenshot.jpeg?tl_px=928,518&br_px=3030,1693&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=771,327)


15. Navigate to <https://docs.langdock.com>


16. Click "Assistant API"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/555826fa-e3fa-41b6-84dd-0372271774e6/ascreenshot.jpeg?tl_px=0,0&br_px=3027,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=197,487)


17. Click "Try it"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/84330482-995a-4959-93f9-daaa5effea22/ascreenshot.jpeg?tl_px=560,0&br_px=3587,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=584,212)


18. Enter the API Key from your clipboard

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/0d92ea18-e220-4986-b00e-e92f230eb4f4/user_cropped_screenshot.jpeg?tl_px=241,0&br_px=3268,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,198)


19. Navigate back to the Langdock application and copy the Assistant ID from the URL. The URL is structured as follows: [https://app.langdock.com/assistants/[ASSISTANT ID]/edit](https://app.langdock.com/assistants/04fd4d3b-08e4-48fe-9055-ee642443248a/edit). You only need the Assistant ID.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/ca5b33ab-e65c-494c-8688-f69357b65db9/ascreenshot.jpeg?tl_px=0,0&br_px=3027,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=365,125)


20. Switch to tab Assistant API - Documentation"


21. Enter the Assistant ID

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/e5a0fda7-5c91-4458-a3a2-b41a6a45e1e9/user_cropped_screenshot.jpeg?tl_px=19,135&br_px=3046,1826&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,277)


22. Add a message with a content and the role "user"

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/34f4e418-98cb-4721-a464-18bc1b7ac57a/user_cropped_screenshot.jpeg?tl_px=50,468&br_px=3077,2160&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=524,403)


23. Click "Send". You should receive a response from the Assistant API with the status code 200. If your API Key is invalid, the status code will be 401.

![](https://ajeuwbhvhr.cloudimg.io/colony-recorder.s3.amazonaws.com/files/2025-01-15/f6694945-9fae-422f-ab75-93aaaf7e3d91/user_cropped_screenshot.jpeg?tl_px=812,0&br_px=3840,1691&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=841,32)
