---
title: '/completion Endpoint'
description: 'Documentation for the Completion Endpoint.'
---

The completions endpoint allows you to complete a prompt with the selected model. To configure the model, you can use the `model` as well as the `provider` parameter. More models will be added over time.

<Info>
  To use the API you need an API key. To request access, please contact us at
  [support@langdock.com](mailto:support@langdock.com)
</Info>
