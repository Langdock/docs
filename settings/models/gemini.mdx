---
title: 'Gemini Setup'
description: 'This is a short step by step guide how to set up Gemini models through Google Vertex.'
---

1. Enable the Vertex AI API in your [Google Cloud Platform](https://console.cloud.google.com/flows/enableapi?apiid=aiplatform.googleapis.com
).

2. Go to "Service Accounts" in the Google Cloud Console IAM Settings

![](/images/vertex-1.png)

3. Click on "Create Service Account"

4. Give the Service Account a name

![](/images/vertex-3.png)

5. Assign the "Vertex AI User" Role

![](/images/vertex-4.png)

6. Create the Service Account.

![](/images/vertex-5.png)

7. You are brought back to the Service Account overview.

![](/images/vertex-6.png)

8. On the overview page, click on "Manage keys"

![](/images/vertex-7.png)

9. Create a new JSON key

![](/images/vertex-8.png)

10. Download & open the JSON file

![](/images/vertex-9.png)

11. Copy the "private\_key" attribute into the "API Secret" field in the Langdock Models page (for example: `-----BEGIN PRIVATE KEY----- {{ private key }} -----END PRIVATE KEY-----\n`)

12. Copy the "client\_email" attribute into the "Endpoint" field in Langdock

13. Use a model ID from the Vertex portal (eg. `gemini-1.5-flash-001` or `gemini-1.5-pro-001`)
